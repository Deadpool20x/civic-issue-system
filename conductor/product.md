# Civic Issue Reporting System

## Product Vision
Empowering citizens by providing a transparent, efficient, and AI-driven platform to report and track civic issues in their city. Enabling smarter city management through role-based access control and automated workflows.

## Core Features
### Implemented
- **AI-Driven Reporting**: Automatic issue detection and categorization from images.
- **Role-Based Access Control**: 5-role hierarchy (CITIZEN, FIELD_OFFICER, DEPARTMENT_MANAGER, MUNICIPAL_COMMISSIONER, SYSTEM_ADMIN).
- **Ward System**: Data scoped to 16 wards across North and South Zones.
- **Premium Dark Theme**: Obsidian & Gold aesthetic across all dashboards.
- **Auto-Assignment**: Issues automatically routed based on category and location.

### Phase 4 (Planned)
- Video upload support for evidence.
- Voice input for issue descriptions.
- Multilingual interface (English, Hindi, Gujarati).
- Citizen feedback loop (Confirm Fixed / Request Reopen).
- Location-based awareness (Nearby issues mini-map).

## Success Metrics
- Resolution time for civic issues.
- User engagement (reports per citizen).
- Data accuracy (AI detection success rate).
- Citizen satisfaction (confirmed fixes).
